# Char Engine

AI SaaS backend prototype built with TypeScript, Express, PostgreSQL, Prisma, JWT authentication, role-based authorization, and OpenAI-ready routes.

## Features

- User registration
- User login
- Password hashing with bcrypt
- JWT authentication
- Protected routes
- Role-based access control
- PostgreSQL database
- Prisma ORM
- AI prompt endpoint
- AI conversation history storage

## Tech Stack

- Node.js
- TypeScript
- Express
- PostgreSQL
- Prisma
- JWT
- bcrypt
- OpenAI SDK
